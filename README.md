# contador-de-cliques
Projeto Devstart

# Propósito do Projeto

Este projeto consiste em uma aplicação simples desenvolvida em React que implementa um contador de cliques.  
A cada vez que o usuário clica no botão, o valor do contador é incrementado em uma unidade.

O objetivo do exercício é praticar os conceitos básicos do React, como componentes funcionais, estado e eventos.


# Tecnologias Utilizadas

- React
- JavaScript
- HTML
- CSS

Clone o repositório:
   ```bash

Acesse a pasta do projeto:
cd contador-de-cliques

npm install

Execute a aplicação:
npm start

http://localhost:3000
